## Crypt是VS2015下的Project，实现了各种加密方法
### AES
用C++实现的一个AES加密类，源代码在\Crypt\Crypt目录下的AES.h 和 AES.cpp <br>
AES加密多个states有4种方案，分别为： <br>
+ ECB模式（电子密码本模式：Electronic codebook） <br>
+ CBC模式（密码分组链接：Cipher-block chaining）未实现 <br>
+ CFB模式（密文反馈:Cipher feedback）未实现 <br>
+ OFB模式（输出反馈：Output feedback）未实现 <br>
### base64
用C++实现的一个base64编码类，源代码在\Crypt\Crypt目录下的base64.h 和 base64.cpp <br>
只实现了编码，解码还没写 <br>
